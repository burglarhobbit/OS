1. Download the code and save it to a text file, say conditions.c. 2. Execute the following command to compile it. %gcc -lpthread conditions.c -o conditions 3. Execute the executable conditions that gcc just created. %conditions Note that % is the prompt from the shell interpreter. You do not need to type it in. Check the printout on the screen and see what's there. You may need to check the source code in order to understand what's happened. Execute the following command to see how you can play gcc. %man gcc
