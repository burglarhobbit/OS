1. Download the code and save it to a text file, say hello.c. 2. Execute the following command to compile it. %gcc -lpthread hello.c -o hello 3. Execute the executable hello that gcc just created. %hello Note that % is the prompt from the shell interpreter. You do not need to type it in. Check the printout on the screen and see what's there. You may need to check the source code in order to understand what's happened. For multithreading in Linux, please check the relevant man pages. Similar functions should be provided. Execute the following command to see how you can play gcc. %man gcc
